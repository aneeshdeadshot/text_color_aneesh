Text Selection Color plugin helps to change the color of the selected text in the website easily.

== Description ==

Text Selection Color plugin helps to change the color of the selected text in the website easily.

Features:

* User can change text color and text background color from plugin settings page with color picker.
* Changes can be seen as preview in the plugin settings page.


== Installation ==

If you have ever installed a WordPress plugin, then installation will be pretty easy:

1. Download the Text Selection Color plugin archive and extract the files
2. Copy the resulting text-selection-color directory into /wp-content/plugins/
3. Activate the plugin through the 'Plugins' menu of WordPress
4. Go to Settings > Text Selection Color page to change colors.
